# Baekji

BaekJi (백지) is a 'blank paper' in Korean.
Simple and minimal writing tool for any platform, including desktop and mobile.

## 소개

백지는 최소한의 구성을 가진 집필용 도구입니다.
문서 양식이 거의 필요 없는 글쓰기를 위한 환경을 제공합니다.

### 목표 기능

- 프로젝트 - 그룹 - 시트 구조의 글 관리
- 간단한 마크다운 기반 리치 텍스트 편집기
- 손쉬운 글 내보내기 및 공유
- 설치 불필요, 데스크탑이든 모바일이든 브라우저만 있다면 사용 가능
- PWA 지원으로 오프라인 앱처럼 사용 가능

### 목표가 아닌 기능

- 복잡한 양식이 필요한 글쓰기 및 일반적인 마크다운 전체 지원
- 협업, 미디어 관리 등
- 다이어그램, 표 등 글쓰기 외 기능

---

## 사용법

### 글 관리

'백지' 에서는 내부적으로 모든 글을 **프로젝트 - 그룹 - 시트** 구조로 관리합니다.

- **프로젝트**: 최상위 단위로, 여러 그룹과 시트를 포함합니다.
- **그룹**: 여러 시트나 그룹을 포함하는 일종의 폴더입니다.
- **시트**: 하나의 글 조각입니다. 마크다운으로 작성 및 저장됩니다.

그룹과 시트는 컴퓨터의 폴더와 파일과 비슷합니다. 트리로 구성되며, 자유롭게 그룹 내에 시트를 추가할 수 있습니다. 단, 그룹과 시트는 자동으로 정렬되지 않습니다. 마우스로 끌거나, 터치 환경에서는 꾹 눌러 재정렬할 수 있습니다.

모든 데이터는 브라우저에 자동으로 저장되며, 브라우저에서 데이터 삭제를 하지 않는 이상 영구적으로 보존됩니다.

### 사이드바

화면 좌측 상단의 버튼으로 사이드바를 열고 닫을 수 있습니다. 단축키는 `Ctrl+\` (macOS: `Cmd+\`) 입니다.

사이드바 상단에서 **프로젝트 목록**과 **현재 프로젝트 트리** 사이를 전환할 수 있습니다.

트리에서 각 항목을 눌러 편집기나 그룹 페이지로 이동할 수 있으며, 우측의 `...` 버튼을 눌러 이름 변경, 색상 지정, 이동, 삭제 등을 수행할 수 있습니다.

여러 항목을 한 번에 삭제하고 싶다면 트리 우측 상단의 **목록 보기** 버튼을 눌러 다중 선택 모드로 전환할 수 있습니다.

### 프로젝트

프로젝트 목록 화면에서 새 프로젝트를 만들거나 기존 백업 파일(`.json`)을 불러올 수 있습니다.

프로젝트는 서로 독립적으로 관리되며 영향을 주지 않습니다.

**백업**: 프로젝트의 모든 데이터를 하나의 `.json` 파일로 내보낼 수 있습니다. 여러 기기에서 사용하려면 이 파일을 클라우드 저장소에 올려 두고 각 기기에서 불러오는 방식으로 활용할 수 있습니다.

### 그룹

그룹을 누르면 포함된 시트와 하위 그룹 목록을 카드 형식으로 볼 수 있습니다.

- **내보내기**: 그룹 내 모든 시트를 하나의 글로 합쳐서 마크다운, HTML, 일반 텍스트 형식으로 내보내거나 공유할 수 있습니다. 내보내는 순서는 트리의 순서를 따릅니다.
- **파일에서 불러오기**: `.md`, `.txt` 파일을 시트로 가져오거나 `.json` 백업 파일에서 하위 트리 전체를 가져올 수 있습니다.
- **분석**: 그룹 내 시트들의 글자 수, 단어 수 등 통계를 한눈에 확인할 수 있습니다.

그룹 이름은 트리 구분용으로만 사용됩니다. 내보내기 결과물에는 반영되지 않습니다. 이름이 `.`으로 시작하는 그룹은 숨김 그룹으로 처리되어 내보내기 시 기본적으로 제외됩니다.

### 시트

시트는 다른 프로그램의 '파일'에 해당합니다. 시트에는 별도의 이름이 없으며, 내용의 첫 줄을 발췌하여 이름 대신 화면에 표시됩니다.

시트는 리치 텍스트 편집기를 사용하지만 폰트, 글자 크기 등 세부 양식은 지원하지 않습니다. 일부 마크다운 호환 서식(굵은 글씨, 기울임, 제목, 목록, 인용, 코드 블록, 취소선)만 지원합니다.

**서식 입력 방법**은 두 가지입니다.

1. 상단 **툴바** 버튼 사용
2. 마크다운 단축 입력 (설정에서 활성화한 항목에 한함):
   - `# ` → 제목 1, `## ` → 제목 2, `### ` → 제목 3
   - `- ` 또는 `* ` → 불릿 목록, `1. ` → 번호 목록
   - `**굵게**`, `*기울임*`, `~~취소선~~`, `` `코드` ``
   - `> ` → 인용, ` ``` ` → 코드 블록

편집기 하단에는 저장된 시각과 글자 수·단어 수 통계가 표시됩니다. 이 영역을 클릭하면 분석 페이지로 이동합니다.

파일을 열면 마지막으로 편집하던 위치(커서)가 복원됩니다.

**자동 저장**: 입력이 멈춘 뒤 일정 시간이 지나면 자동으로 저장됩니다. 수동으로 저장하려면 `Ctrl+S` (macOS: `Cmd+S`)를 누릅니다.

### 검색

사이드바 상단 또는 프로젝트 내에서 검색 기능을 사용할 수 있습니다. 현재 프로젝트 내 모든 시트의 본문을 대상으로 검색하며, 대소문자 구분과 정규식 검색을 지원합니다.

### 분석

그룹 또는 시트에서 분석 페이지로 이동하면 각 시트의 글자 수, 단어 수, 바이트 크기 등을 표로 확인할 수 있습니다. 공백 포함 여부를 전환할 수 있습니다.

### 설정

설정에서 다음 항목을 조정할 수 있습니다.

- **테마**: 라이트/다크 각각 기본(Default), 웜(Warm), 쿨(Cool) 중 선택
- **타이포그래피**: 글꼴(산세리프, 세리프, 고정폭, 시스템, 커스텀), 에디터·미리보기 글자 크기, 줄 간격, 첫 줄 들여쓰기
  - Noto 폰트를 사용하면 한글을 포함한 다국어 글꼴을 지원하지만 최대 30MB 다운로드가 필요합니다.
- **자동 저장 간격**: 입력 멈춤 후 저장까지의 대기 시간(초)
- **마크다운 입력 규칙**: 단축 입력으로 변환할 마크다운 요소 선택

### 단축키

| 단축키                   | 동작               |
| ------------------------ | ------------------ |
| `Ctrl+S` / `Cmd+S`       | 저장               |
| `Ctrl+\` / `Cmd+\`       | 사이드바 열기/닫기 |
| `Ctrl+Z` / `Cmd+Z`       | 실행 취소          |
| `Ctrl+Y` / `Cmd+Shift+Z` | 다시 실행          |
| `Ctrl+B` / `Cmd+B`       | 굵게               |
| `Ctrl+I` / `Cmd+I`       | 기울임꼴           |

### 색상 지정

그룹과 시트에는 색상을 지정할 수 있습니다. 색상은 트리와 카드 목록에서 시각적으로 구분하는 용도로 사용됩니다.

### 숨김 그룹

이름이 `.`으로 시작하는 그룹은 숨김 처리됩니다. 트리에서는 보이지만 내보내기 시 기본적으로 제외되며, 옵션을 통해 포함시킬 수 있습니다. 초고, 메모, 참고 자료 등을 별도로 관리할 때 유용합니다.

### PWA (앱으로 설치)

'백지'는 PWA(Progressive Web App)를 지원합니다. 브라우저에서 '홈 화면에 추가' 또는 '앱 설치' 옵션을 선택하면 오프라인에서도 사용할 수 있는 독립 앱처럼 실행됩니다.
