# The Editor

The editor supports **Markdown**-style formatting as you type.

## Inline styles

- `**bold**` → **bold**
- `*italic*` → _italic_
- `~~strikethrough~~` → ~~strikethrough~~
- `` `code` `` → `code`

## Headings

Start a line with `#`, `##`, or `###` followed by a space.

## Lists

- Start a line with `-` or `*` for a bullet list.
- Start with `1.` for a numbered list.
- Use **Tab** / **Shift+Tab** to indent or dedent items.

## Other blocks

- Start a line with `>` for a blockquote.
- Start a line with ` ``` ` for a code block.

## Toolbar

The toolbar at the top gives quick access to formatting, links, images, and more. On mobile, it stays pinned above the keyboard.

## Keyboard shortcuts

- `Ctrl / ⌘ + S` — Save
- `Ctrl / ⌘ + Z` — Undo
- `Ctrl / ⌘ + Y` — Redo
- `Ctrl / ⌘ + B` — Bold
- `Ctrl / ⌘ + I` — Italic
