# Customizing Baekji

Baekji keeps settings minimal, but there are a few things worth knowing.

## Typography

In **Settings**, you can adjust:

- Font family (sans-serif, serif, monospace, or a custom font name)
- Editor font size
- Line height
- First-line indent

## Themes

Choose between light and dark mode themes — Default, Warm, or Cool. The theme follows your system preference by default.

## Markdown input rules

You can toggle which Markdown shortcuts are active while typing — headings, lists, inline styles, blockquotes, code blocks, smart quotes, and more.

## Autosave

Your work is saved automatically. You can adjust the autosave interval in Settings. Press **Ctrl/⌘ + S** at any time to save immediately.
